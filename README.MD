# Herdtech-app – Site Web d'agriculture

## Description du projet

**Herdtech-app** est un site web d'agriculture moderne, élégant et entièrement responsive, réalisé exclusivement avec **HTML**, **CSS** et **Bootstrap**. Le projet consiste à reproduire fidèlement l’apparence et la structure d’un site d'agriculture professionnel comprenant plusieurs pages clés : Accueil, Menu, À propos et Réservation.

---

## Objectifs du projet

* Créer un site web de d'agriculture **responsive** (mobile, tablette, desktop)
* Utiliser uniquement **HTML5**, **CSS3** et **Bootstrap 5**
* Appliquer les principes de **design moderne et propre**
* Structurer le site avec plusieurs pages fonctionnelles
* Intégrer une **police professionnelle** adaptée à l’identité de l'agriculture

---

## Technologies utilisées

* **HTML5** – structure des pages
* **CSS3** – personnalisation du design
* **Bootstrap 5** – mise en page responsive et composants UI
* **Google Fonts** – typographie du site
